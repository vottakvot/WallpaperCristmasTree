# WallpaperCristmasTree
Android NDK. OpenGL ES 2.0.  Cristmas tree live wallpaper.

<u><b>Play market:</u></b> https://play.google.com/store/apps/details?id=ru.testsimpleapps.lifewallpapertree

  - has 3 shader programs for:
    * snow
    * ellipse
    * star
